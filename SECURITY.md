# Security Policy

## Reporting a Vulnerability
Please report security issues privately via email to: baltasar.collantesginer@gmx.es
Do not open public issues with exploit details.

## Scope
This project handles identity and privileged access. Treat all secrets as sensitive.
Avoid static credentials and ensure short-lived tokens in demos and code.

## Responsible Disclosure
We follow coordinated disclosure. We will acknowledge reports and provide timelines for fixes when feasible.
